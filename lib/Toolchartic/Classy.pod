=head1 NAME

Toolchartic::Classy - Class-loading support for Toolchartic Perl resources

=head1 SYNOPSIS

  use Toolchartic::Classy qw(
    # add exported functions here
  );

=head1 DESCRIPTION

Toolchartic::Classy provides support for loading class-oriented
Toolchartic Perl resources.

These resources are not ordinary Perl modules loaded through Perl's
native module system.

Instead, they are loaded through Toolchartic machinery under ad-hoc
package names and are expected to follow the contract documented by
Toolchartic::Classy.
